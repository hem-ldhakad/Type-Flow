# Product Requirements Document

## Product
TypeSprint

## Goal
Build a production-ready multiplayer typing platform inspired by Monkeytype.

## Users
- Students
- Developers
- Competitive typists

## MVP
- Register/Login
- JWT Authentication
- Dashboard
- Create Room
- Join Room
- Countdown
- Shared Paragraph
- Real-time Race
- WPM
- Accuracy
- Match Results
- Match History

## Future
- Ranked Mode
- Friends
- XP
- Themes
- Spectator Mode
- Achievements

## Tech Stack
React (JavaScript), Express, PostgreSQL, Prisma, Socket.IO.
